About


Being a BCIT set rep can be difficult. On top of the regular course load, you are also responsible for keeping all the students in the set up to date with the latest BCIT news. Our set rep, Jose, has taken on an additional role of updating everyone on when course assignments are due. This can potentialy increase the amount of stress the set reps have to endure. THe purpose of the set-rep-discord bot not to replace the set rep, but rather to make their life easier. By automating due date reminders, all set reps have to do is input what and when an assignment or quiz is due and the bot will automatically set a reminder announcement for that item.
